# 原生

```html

<div id="#app">
    <p>{{msg}}</p>
</div>

<script>
var vm=new Vue({
    el:"#app",
    data:{
        msg:"1111"
    }
})

</script>
```


# v-cloak

使用`v-cloak` 能够解决插值表达式闪烁问题，只会替换原来的占位符，不会替换原来的内容

```html

[v-cloak]{
    display:none;
}


<div id="#app">
    <p v-cloak="msg" >+++++{{msg}}-------</p>
</div>

<script>
var vm=new Vue({
    el:"#app",
    data:{
        msg:"1111"
    }
})

</script>
```

# v-text

使用`v-text` 渲染，可以完全替换元素中原本的内容

```html

<div id="#app">
    <p v-text="msg" />
</div>

<script>
var vm=new Vue({
    el:"#app",
    data:{
        msg:"1111"
    }
})

</script>

```

# v-html

如果渲染的值含有标签的形式，可以使用`v-html`进行渲染。

```html

<div id="#app">
    <p v-html="msg"></p>
</div>

<script>
var vm=new Vue({
    el:"#app",
    data:{
        msg:"<p>123</p>"
    }
})

</script>

```
# v-bind

`v-bind`是Vue提供的用于绑定属性的指令

```html

<div id="#app">
    <input type="button" value="按钮" b-bind:title="msg" />

    <input type="button" value="按钮" b-bind:title="msg+'这里是追加的东西'" />

    <!-- 或者写成这样 只需要写个 : 即可-->
    <input type="button" value="按钮"  :title="msg+'这里是追加的东西'" />
</div>


<script>
var vm=new Vue({
    el:"#app",
    data:{
        msg:"自定义的title"
    }
})

</script>

```

# v-on

`v-on`是用来绑定事件机制,缩写是 @

```html

<div id="#app">
    
    <input type="button" value="按钮" v-on:click="show" />

    <!-- 或者写成这样 只需要写个 @ 即可-->
    <input type="button" value="按钮"  @click="show" />
</div>


<script>
var vm=new Vue({
    el:"#app",
    data:{
        msg:"自定义的title"
    },
    methods:{
        show:function(){
            alert('被点击了')
        }
    }
})

</script>

```

# 跑马灯实例

```html

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="./vue.js"></script>
    <!-- <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> -->
    <title>跑马灯实例</title>
</head>

<body>
    <div id="app">
        <input type="button" value="浪起来" @click="lang"></input>
        <input type="button" value="不浪" @click="nolang"></input>

        <h1 v-text="msg"></h1>
    </div>

    <script>
        // 在VM实例中，如果想要获取data的数据,或者methods里面的方法，必须通过this.数据属性名 或者 this.方法名来进行访问
        var vm = new Vue({
            el: '#app',
            data: {
                msg: '猥琐发育,别浪！',
                intervalId: null //定义一个定时器id
            },
            methods: {
                // ES6写法
                lang() {
                    // 如果没有使用箭头函数要注意this指向问题
                    // var _this = this

                    // 防止开启多个定时器
                    if (this.intervalId != null) return;
                    // 开启监听器,并使用箭头函数（内部就可使用外部的this，解决this指向问题）
                    this.intervalId = setInterval(() => {
                        // alert(_this.msg)
                        // 注意:VM会监听data中所有的数据，如果发生改变会自动发生变化，并同步到页面
                        var start = this.msg.substring(0, 1);
                        var end = this.msg.substring(1);
                        // 重新给msg赋值
                        this.msg = end + start
                    }, 400)

                },
                nolang() {
                    // 点击停止定时器
                    clearInterval(this.intervalId);
                    // 每次清除定时器，重新赋值为null
                    this.intervalId = null;
                }
            },
        })
    </script>
</body>

</html>
```

# 事件修饰符

- .stop  阻止冒泡
- .prevent 阻止默认事件
- .capture 事件捕获
- .self 只发生该元素身上，触发时触发回调
- .once 事件只发生一次

```html

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="./vue.js"></script>
    <title>事件修饰符</title>
    <style>
        .inner {
            height: 150px;
            background-color: aqua;
        }
    </style>
</head>

<body>
    <div id="app">
        <!-- 使用 .stop 阻止冒泡事件 (从里到外)-->
        <!-- <div class="inner" @click="Hader">
            <input type="button" value="点他" @click.stop="btn">
        </div> -->


        <!--使用  .capture 实现捕获事件 (从外到里 )-->
        <!-- <div class="inner" @click.capture="Hader">
            <input type="button" value="点他" @click="btn">
        </div> -->


        <!--使用 .self 实现自身触发事件，只有自己游行才会触发，不受其他的影响  -->
        <!-- <div class="inner" @click.self="Hader">
            <input type="button" value="点他" @click="btn">
        </div> -->



        <!-- 使用  .prevent  实现阻止默认行为 (就是点击不触发，也就是不跳转链接)-->
        <!-- <a href="http://www.baidu.com" @click.prevent="link" target="_blank">百度一下</a> -->


        <!-- 使用  .once 实现只是用一次阻止默认行为，当第二次的时候不生效 -->
        <!-- <a href="http://www.baidu.com" @click.prevent.once="link" target="_blank">百度一下</a> -->


    </div>
    <script>
        var vm = new Vue({
            el: '#app',
            data: {

            },
            methods: {
                Hader() {
                    console.log('触发背景');

                },
                btn() {
                    console.log('触发按钮');
                },
                link() {
                    console.log('触发链接的点击');
                }
            },
        })
    </script>

</body>

</html>
```

# v-model和双向数据绑定

```html

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>双向数据绑定</title>
    <script src="./vue.js"></script>
</head>

<body>
    <div id="app">
        <h4>{{msg}}</h4>
        <!-- v-bind 只能实现数据的单向绑定，从M自动绑定到V -->
        <input type="text" :value="msg" />

        <!-- v-model 能实现数据表单元素和Model中的数据的双向数据绑定 -->
        <input type="text" v-model="msg" />
    </div>


    <script>
        var vm = new Vue({
            el: '#app',
            data: {
                msg: 'hello~~~~'
            },
            methods: {

            },
        })
    </script>
</body>

</html>
```
