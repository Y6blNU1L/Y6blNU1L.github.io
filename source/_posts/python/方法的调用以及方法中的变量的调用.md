## 方法的调用以及方法中的变量的调用

``` python
class test:
    def __init__(self):
        self.d=6

    def b(self):
        self.f=5
        print("初始",self.d) #直接调用init里面的值

    def c(self):
        s = self.f
        print(s) #打印b方法的self.f的值

        self.b()#方法之间的调用

if __name__ == '__main__':
    t=test()#初始化实例类
    t.b()
    t.c()
```
