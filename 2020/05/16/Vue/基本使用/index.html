Vue

原生

v-cloak

使用v-cloak 能够解决插值表达式闪烁问题，只会替换原来的占位符，不会替换原来的内容

v-text

使用v-text 渲染，可以完全替换元素中原本的内容

v-html

如果渲染的值含有标签的形式，可以使用v-html进行渲染。

v-bind

v-bind是Vue提供的用于绑定属性的指令

v-on

v-on是用来绑定事件机制,缩写是 @

事件修饰符

  • .stop 阻止冒泡
  • .prevent 阻止默认事件
  • .capture 事件捕获
  • .self 只发生该元素身上，触发时触发回调
  • .once 事件只发生一次
